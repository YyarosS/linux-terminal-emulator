# linux-terminal-emulator
### ИКБО-30-24 вариант №10
### Операционнаая система: Линукс

## Общее описание
Эта **CLI** программа является эмулятором для языка оболочки ОС. Данный эмулятор должен быть макисмально похож на консольную строку **UNIX**-подобных ОС. Большинство функций в нем пока
представляют собой заглушки, но диалог с пользователем уже поддерживается.
## Описание всех функций и настроек
  - printCommandWithArguments -- функция вывода команд cd или ls, которые были задействованы и вывод их аргументов
  - checkSystemVariable -- функция проверки существования вводимой системной переменной в ОС
  - executeCommand -- функция выполнения команд и вывод их результата
  - parseSystem -- парсер, который поддерживает раскрытие переменных
  - exit -- команда выхода из программы
окружения реальной ОС 
  - Если в консоль была введения переменная, которой нет в ОС, то консоль выдает пустую строку
  - Если ничего не было ввведено в консоль, то повторно выводится системное приглашение
  - Если программа не поддерживает введенную команду, то программа сообщает об ошибке выполнения команд
## Описание команд для сборки проекта и запуска тестов
Для выполнения сборки проекта и запуска тестов, необходимо удостоверить в наличии компилятора **g++**. После скопировать репозиторий, в терминале зайти в каталог, куда была установлена программа и выполнить команду:
```bash
g++ -o main main.cpp
```
После сборки можно открыть программу командой:
```bash
./"main"
```
